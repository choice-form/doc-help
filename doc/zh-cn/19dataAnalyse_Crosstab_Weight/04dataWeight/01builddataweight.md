# 创建权重方案

我们有3中权重设置方案：
+ 倾斜加权（边际目标）：也被称为迭代比例拟合或随机迭代法。它适用的情况是：您需要调整采样群体以满足目标群体的代表，但不知道多重特征群体的比例。
+ 基于单元格的（动态联锁回复）：加权不重叠。您为总计 100% 的每个回复组指定目标百分比。
+ 静态（联锁回复）：加权不重叠。您指定应用于每个回复组的权重或乘数。
